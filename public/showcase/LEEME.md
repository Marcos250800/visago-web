# Fotos del carrusel (Destinos & Recursos)

Coloca aquí las fotos con **estos nombres exactos** (se sirven en `/showcase/...`):

- `espana.jpg` — foto para la tarjeta de España (p. ej. la Alhambra)
- `eeuu.jpg` — foto para la tarjeta de EE.UU. (skyline)
- `becalab.jpg` — foto para la tarjeta de BecaLab

Formato: **JPG** (o PNG/WebP). Apaisada, mínimo ~1600px de ancho.

La web les aplica automáticamente **desaturación parcial** (≈55% gris, término medio
color/B-N) + un degradado para que el texto se lea. Si una foto no está, la tarjeta
muestra un degradado de marca (no se rompe nada).

> Para ajustar el nivel de color/desaturación, dímelo y cambio el filtro.
